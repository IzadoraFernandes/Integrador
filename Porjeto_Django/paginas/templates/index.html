Um Software de Gerenciamento Laboratorial


Sobre o Sistema


Desenvolvido com objetivo gerenciar salas de Laboratório, o GL permite que os usuários consigam reservar salas de forma mais fácil e prática.

Nele há informações como: tipo de laboratório, sala, horário e data da reserva, nome do docente, disciplina, tema da aula e turma.

Funcionalidades das reservas

Confira os principais recursos da solução.

Adicionar reservas

Será possível ser escolhido um laboratório para realizar o agendamento com horário e dia definido para uma determinada turma

Modificar reservas

Há possibilidade de fazer modificação em alguma informação sobre uma reserva que já esteja cadastrada no sistema, como: data, horário e descrição.

Cancelar reservas

Poderá fazer o cancelamento de uma reserva de laboratório, tornando assim o lugar disponível para outro docente ocupar.
